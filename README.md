## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/gaxzbi/Repositorio_entra21_python.git
```

Entre no diretório do projeto

```bash
  cd my-project
```

Instale as dependências

```bash
  pip install requirements.txt
```

Inicie o servidor

```bash
  python app.py
```
